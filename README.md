# pretext-table

Work in progress. A high-performance virtual-scrolling table viewer powered by [@chenglou/pretext](https://github.com/chenglou/pretext) for DOM-free row height calculation, with [Apache Arrow](https://arrow.apache.org/) for columnar data and [Semiotic](https://semiotic.nteract.io/) for column summary charts.

```sh
npm install
npm run generate
npm run dev
```
